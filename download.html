How to Install

Step-by-step installation instructions for each platform

Android Installation Guide

1
Enable Unknown Sources

Go to Settings ‚Üí Security ‚Üí Enable "Install from Unknown Sources"

2
Download APK File

Click the "Download APK" button above and save the file

3
Install Application

Open the downloaded APK file from your notifications or file manager

4
Grant Permissions

Allow storage permissions when prompted for saving files

System Requirements

  • Android 5.0 (Lollipop) or higher
  • Minimum 20 MB free storage
  • Any ARM/x86 processor

Windows Installation Guide

1
Download Installer

Click "Download Installer" button and save the .exe file

2
Run Installer

Double-click the downloaded file. If Windows SmartScreen appears, click "More info" ‚Üí "Run anyway"

3
Choose Install Location

Select installation directory or use default path

4
Complete Installation

Follow on-screen instructions. Launch from desktop shortcut or Start menu

System Requirements

  • Windows 7/8/10/11 (64-bit)
  • Minimum 100 MB free space
  • .NET Framework 4.5 or higher
